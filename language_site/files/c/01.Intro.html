Facts related to functions :

1. In a function arguments are always passed by value. Pass by reference is simulated by c by passing pointer values.

2. Functions can return any type except arrays and functions. This limitation can be overcome by returning pointer to an array or pointer to function.

3. Name of the function is actually a pointer variable to the function
    example :

    int main(){
        printf("%u",main); // prints address of main
        return 0;
    }

4. Declaration of your user defined function should be before main function.

5. In c three continuous dots is known as ellipsis which is variable number of arguments of function. The values to parameters are assigned one by one.
    example:

    
#include<stdio.h> 
  
void dynamic(int s, ...) 
{ 
    printf("%d ", s); 
} 
  
int main() 
{ 
    dynamic(2, 4, 6, 8); 
    dynamic(3, 6, 9); 
    return 0; 
}
